# Guia de Uso - SAR-EB

## Bem-vindo ao SAR-EB

Este é um guia rápido para navegar e utilizar o Sistema de Apoio a Requisições do Exército Brasileiro.

## Acesso ao Sistema

### Tela de Login
1. Acesse a página inicial (`/`)
2. Digite seu usuário e senha
3. Clique em "Entrar"

**Primeira vez?**
- Clique em "Cadastre-se" para criar uma nova conta
- Esqueceu a senha? Use "Esqueceu sua senha?" para recuperar

## Navegação Principal

### Sidebar (Menu Lateral)
O menu lateral está sempre visível e contém:
- **Dashboard** - Visão geral do sistema
- **Requisições** - Gerenciamento de requisições
- **Checklist** - Gerenciamento de checklists
- **Certidões** - Visualização de certidões
- **Perfil** - Suas informações pessoais
- **Sair** - Logout do sistema

**Mobile:** No celular, clique no ícone ☰ (hambúrguer) para abrir o menu.

## Funcionalidades Principais

### 1. Dashboard
**O que você vê:**
- Estatísticas gerais (requisições ativas, checklists, certidões)
- Atalhos rápidos para criar requisição, checklist e ver certidões
- Lista de requisições recentes com status

**Ações rápidas:**
- Clique em "Nova Requisição" para criar uma requisição
- Clique em "Criar Checklist" para iniciar um checklist
- Clique em "Ver Certidões" para acessar as certidões

### 2. Requisições

#### Listar Requisições
- Visualize todas as requisições
- Use a busca para encontrar requisições específicas
- Filtre por status ou tipo de empenho
- Clique em "Ver Detalhes" para mais informações

#### Criar Requisição
1. Clique em "Nova Requisição"
2. Selecione o **Tipo de Empenho**:
   - Ordinário
   - Global
   - Estimativo
3. Preencha os campos:
   - Título da Requisição
   - Unidade Requisitante
   - Setor
   - Descrição detalhada
   - Valor Estimado
   - Prazo de Entrega
   - Justificativa
   - Observações (opcional)
4. Clique em "Enviar Requisição" ou "Salvar Rascunho"

**Dica:** O modelo de requisição é gerado automaticamente com base no tipo de empenho selecionado.

#### Detalhes da Requisição
- Visualize todas as informações
- Acompanhe o histórico/timeline
- Baixe o PDF da requisição
- **Administradores:** Aprove ou rejeite requisições

### 3. Checklist

#### Listar Checklists
- Veja todos os checklists criados
- Visualize o progresso de cada um (porcentagem de conclusão)
- Acesse recursos adicionais:
  - **Modelos de Processo**: Documentos prontos (DOCX, XLSX)
  - **Fundamentação Legal**: Legislação aplicável

#### Criar Checklist
1. Clique em "Criar Checklist"
2. Selecione o **Tipo de Empenho**
3. Dê um título ao checklist
4. Os itens padrão são carregados automaticamente
5. Marque/desmarque os itens desejados
6. Adicione itens personalizados se necessário
7. Clique em "Criar Checklist"

#### Acompanhar Checklist
- Marque itens conforme forem concluídos
- Veja o progresso em porcentagem
- Baixe o PDF do checklist

#### Modelos de Processo
- Visualize modelos disponíveis
- Filtre por tipo de empenho
- Baixe documentos prontos:
  - Termo de Referência (DOCX)
  - Justificativa Técnica (DOCX)
  - Pesquisa de Preços (XLSX)
  - Cronograma de Entregas (XLSX)
  - E mais...

#### Fundamentação Legal
- Consulte a legislação aplicável:
  - Lei nº 14.133/2021 (Nova Lei de Licitações)
  - Decreto nº 11.462/2023
  - Lei nº 4.320/1964
  - IN SEGES nº 67/2021
- Expanda cada item para ver detalhes
- Acesse links para textos completos

### 4. Certidões

- Visualize todas as certidões disponíveis
- Veja estatísticas:
  - Total de certidões
  - Certidões válidas
  - Certidões próximas ao vencimento
  - Certidões vencidas
- Status por cores:
  - 🟢 Verde: Válida
  - 🟡 Amarelo: Próxima ao vencimento (30 dias)
  - 🔴 Vermelho: Vencida
- Tipos de certidões:
  - Federal
  - Estadual
  - Municipal
  - Trabalhista
- Ações:
  - Visualizar certidão
  - Baixar certidão

### 5. Perfil

#### Informações Pessoais
- Nome Completo
- Posto/Graduação
- E-mail
- Telefone
- Unidade
- Setor
- Cargo
- Localização

#### Alterar Senha
1. Digite a senha atual
2. Digite a nova senha
3. Confirme a nova senha
4. Clique em "Alterar Senha"

#### Estatísticas
- Requisições criadas
- Checklists criados
- Aprovações realizadas

#### Atividades Recentes
- Veja suas últimas ações no sistema
- Timeline com data e hora

#### Permissões
- Visualize suas permissões no sistema

## Dicas de Uso

### Navegação
- Use a sidebar para navegar entre módulos
- Use o botão "Voltar" (←) para retornar à página anterior
- Breadcrumbs mostram onde você está

### Busca e Filtros
- Use a barra de busca para encontrar rapidamente
- Combine filtros para resultados específicos
- Limpe os filtros para ver todos os itens

### Status e Badges
- **Verde**: Aprovado/Válido/Concluído
- **Azul**: Em análise/Informativo
- **Amarelo**: Atenção/Próximo ao vencimento
- **Vermelho**: Rejeitado/Vencido/Erro

### Responsividade
- Desktop: Layout completo com sidebar fixa
- Tablet: Layout adaptado
- Mobile: Menu hambúrguer, navegação simplificada

## Ações Administrativas

**Disponível apenas para Administradores:**

### Validar Requisições
1. Acesse os detalhes da requisição
2. Revise todas as informações
3. Use os botões de validação:
   - "Aprovar" (verde)
   - "Rejeitar" (vermelho)
4. Ou altere o status manualmente

### Atualizar Status
1. Nos detalhes da requisição
2. Use o dropdown "Alterar Status"
3. Selecione o novo status
4. Clique em "Atualizar Status"
